# krish-jeswal.web.app

Personal portfolio and landing page.

## Built with

- HTML5 — semantic structure
- CSS3 — custom properties, Grid, Flexbox, animations
- Vanilla JavaScript — IntersectionObserver, DOM manipulation

No frameworks. No build tools. No dependencies.

## Structure
portfolio/
├── index.html
├── style.css
├── script.js
└── assets/
└── Krish_Resume.pdf

## Sections

- **Home** — intro, links
- **Skills** — grouped by category
- **Projects** — IETE-RVCE Website, MoodVue, Synaptic-BioMemristor
- **Timeline** — education and community involvement
- **Contact** — form + socials

## Local development

No setup required. Clone and open `index.html` in a browser.

```bash
git clone https://github.com/KrishJeswal/portfolio
cd portfolio
open index.html
```

## Deployment

Hosted on Firebase. Live at [krishjeswal.web.app](https://krish-jeswal.web.app)
